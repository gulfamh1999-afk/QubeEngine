# QUBE Engine

Graph Spatial Quantum Machine Learning Library

## Installation

pip install qube-engine

## Features

- Graph Descriptor Learning
- Quantum-inspired Features
- Scikit-Learn Compatible
- Benchmarks
- CLI
- Save/Load Models

## Example

from qube import QubeEngine

model = QubeEngine()
model.fit(X, y)
pred = model.predict(X)
